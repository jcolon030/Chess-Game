# main.py overview:

- List of still alive units for both sides
- A set up gameboard with pieces, backup board for undo feature, coordinate  
  plane for tracking
- print menu function 
- starting menu with options tree

- each class contains moving instructions and rules and output the gameboard updated
- with also cover attacks, death, and promotion

# in_game_settings.py overview:

- will assign strings to classes 
- settings can be modified for possible machine learning enchancement
